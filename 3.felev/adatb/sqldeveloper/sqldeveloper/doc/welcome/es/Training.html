Rutas de Formación de Oracle para Desarrollo SQL y PL/SQL
Aprender de Oracle University

Oracle ofrece una completa formación de desarrollo SQL y PL/SQL. Hay dos rutas de formación que representan un buen plan de estudios para los nuevos usuarios o los usuarios experimentados en SQL y PL/SQL. Los cursos de ambas rutas de formación utilizarán SQL Developer como herramienta gráfica para el desarrollo de bases de datos.

En Ruta de Formación SQL, aprenderá las sentencias SQL esenciales que se utilizan la mayor parte del tiempo. También puede obtener conocimientos avanzados sobre el ajuste de sentencias SQL para obtener el máximo rendimiento y el uso de sentencias SQL analíticas para almacenes de datos.

En Ruta de Formación PL/SQL, aprenderá las sentencias PL/SQL esenciales que se utilizan la mayor parte del tiempo y temas de programación avanzada. También obtendrá el conocimiento para escribir programas de aplicaciones seguros.

 

 

Además de cursos de formación presencial, Oracle ofrece formación basada en Internet a través de la Biblioteca en Línea de Oracle University. Este cómodo servicio incluye una matriz de títulos sobre las tecnologías de vanguardia de Oracle.
