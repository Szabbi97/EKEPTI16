Percorsi di formazione Oracle per lo sviluppo SQL e PL/SQL
Apprendimento da Oracle University

Oracle offre una formazione completa sullo sviluppo SQL e PL/SQL. Sono disponibili due percorsi di formazione per un piano di apprendimento efficiente per i principianti o per coloro che hanno già esperienza con SQL e PL/SQL. I corsi di entrambi i percorsi di apprendimento utilizzano SQL Developer come strumento grafico per lo sviluppo del database.

Nel Percorso di apprendimento su SQL vengono illustrate le istruzioni SQL essenziali usate nella maggior parte dei casi. È inoltre possibile acquisire una conoscenza avanzata per sottoporre a tuning le istruzioni SQL per prestazioni ottimali e usare le istruzioni SQL analitiche per il data warehousing.

Nel Percorso di apprendimento su PL/SQL vengono illustrate le istruzioni PL/SQL essenziali usate nella maggior parte dei casi e gli argomenti di programmazione avanzata. È inoltre possibile acquisire le conoscenze necessarie per scrivere programmi applicativi sicuri.

 

 

Oltre a una gamma di corsi in classe, Oracle offre formazione basata su Web tramite la Oracle University Online Library. Questo comodo servizio include una serie di titoli relativi all'apprendimento delle tecnologie di punta Oracle.
