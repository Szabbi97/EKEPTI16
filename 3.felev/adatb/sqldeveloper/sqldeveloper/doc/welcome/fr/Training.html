Parcours de formation Oracle pour le développement SQL et PL/SQL
Apprendre avec Oracle University

Oracle propose une formation complète sur le développement SQL et PL/SQL. Deux parcours de formation sont proposés, qui offrent un programme d'apprentissage efficace pour les utilisateurs débutants ou expérimentés sur SQL et PL/SQL. Les cours de ces deux parcours utilisent SQL Developer comme outil graphique pour le développement de base de données.

Dans le parcours pédagogique SQL, vous découvrez les instructions SQL principales, utilisées la plupart du temps. Vous accédez également à des connaissances avancées quant à l'adaptation des instructions SQL pour obtenir des performances optimales, et quant à l'utilisation des instructions SQL analytiques pour Data Warehouse.

Dans le parcours pédagogique PL/SQL, vous découvrez les instructions PL/SQL principales, utilisées la plupart du temps, ainsi que les sujets de programmation avancés. Vous accédez également à des connaissances relatives à l'écriture de programmes d'application sécurisés.

 

 

En plus de sa gamme de cours en salle, Oracle propose une formation Web via la bibliothèque en ligne d'Oracle University. Ce service pratique inclut un tableau des intitulés d'enseignement des technologies de pointe Oracle.
